Configuration

Wondering how to configure the wallpaper? Well it is quite simple. Open the JSON file located at `@{config_file_location}` and configure the options present. Of course not all options are visable at the start, but the config is mainly what I thought most people would like to edit. Here is an entire list of options:

Option Potential Values/Description
visualizer.ringType This option controls the cental ring around the image. Here are the following values for the type:
    @{bar_types}
bloom.intensity Controls how bright the bloom is. 0 means that bloom will be off, I do not recommend going above 1 as that can cause color banding and/or hurt your eyes.
visualizer.rotSpeed This controls the speed at which the visualizer goes around the image. A negitive value will send it in the opposite direction.
unity.vsync Tells the Unity game engine rather to use VSync or not. It is recommended to leave this on unless you are experensing low framerate.
bloom.enabled This is here for legacy reasons, it does nothing.
vis.embersEnabled Controls if embers will fall down from the top of the screen or not.
vis.bandAmount Controls how many bands are on the screen. Note that the range for this value is [8,128]. Setting this value outside the range will be ignored.
You may note like it, but anything outside this range can cause stability issues, graphical issues, and memory issues. Trust me, don't do it.
logo.imagePath This option changes the logo in the center of the screen. It is recommended to use an image that is 1200x1200, otherwise the image might be either too small or too big. To see if the image will fit, and you have Blender or know how to use it, an Object file is located at `@{obj_file_loc}`. The default scale for images is 100px = 1 Unity Unit, for those who need to know.
